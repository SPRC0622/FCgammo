# 🎉 FC온라인 타이머 v1.0.1 업데이트

## 🆕 새로운 기능

### 1. 🔍 선수 검색 기능 추가
- 선수명 또는 시즌으로 빠르게 검색
- 실시간 검색 결과 표시
- 검색된 선수 수 표시

### 2. 🔄 감독모드 자동 재시작
- 알림 확인 버튼 클릭 시 자동으로 10분 타이머 재시작
- 매번 시작 버튼 누를 필요 없음
- 연속 플레이에 최적화

## 🔧 개선사항

### 시각적 개선
- ✅ 현재 시간 글씨색 흰색으로 변경 (배경과 대비 향상)
- ✅ 텍스트 그림자 추가로 가독성 개선

### 사용성 개선
- ✅ 타이머 기본값 10분 유지
- ✅ 검색창 추가로 많은 선수 관리 용이

## 📦 다운로드

| 파일명 | 설명 | 용량 |
|--------|------|------|
| `FC온라인 타이머 Setup 1.0.1.exe` | 설치 버전 (추천) | ~80MB |
| `FC온라인타이머-portable.exe` | 포터블 버전 | ~80MB |

## 💡 주요 변경사항 요약

### Before (v1.0.0)
- 알림 후 수동으로 타이머 재시작 필요
- 선수 목록에서 직접 찾기
- 현재 시간이 잘 안 보임

### After (v1.0.1)
- ✨ 알림 확인 → 자동 10분 재시작
- ✨ 검색창으로 빠른 선수 검색
- ✨ 현재 시간 선명하게 표시

## 🔄 업데이트 방법

### 기존 사용자
1. 기존 프로그램 제거 (선택사항)
2. 새 버전 다운로드 및 설치
3. 기존 CSV 파일은 그대로 사용 가능

### 신규 사용자
1. exe 파일 다운로드
2. 실행 (Windows Defender 경고 시 "추가 정보" → "실행")
