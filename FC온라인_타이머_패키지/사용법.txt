🎯 FC온라인 감독모드 & 이적시장 타이머 사용법

📁 파일 구성:
- FC온라인_타이머.html (메인 프로그램)
- 선수목록.csv (선수 데이터)
- 사용법.txt (이 파일)

🚀 실행 방법:
1. FC온라인_타이머.html 파일을 더블클릭
2. 브라우저에서 자동으로 열림

⚽ 감독모드 타이머:
- 시작: 10분 카운트다운 시작
- +5분: 연장전용 (15분으로 연장)
- +1분: 미세 조정용
- 리셋: 10분으로 초기화
- 시간 종료시: 팝업 알림

📈 이적시장 알림:
1. "파일 선택" 버튼 클릭
2. "선수목록.csv" 파일 선택
3. 자동으로 알림 설정 완료
4. 갱신 시간마다 자동 알림

📊 선수목록.csv 편집법:
1. 엑셀이나 메모장으로 열기
2. 선수 추가/수정/삭제
3. CSV 형식으로 저장
4. 다시 파일 업로드

CSV 형식 예시:
선수명,시즌,갱신패턴,갱신분
페페 레이나,TB시즌,짝수시,04
메시,22TOTY,홀수시,31
호날두,21UCL,짝수시,08

💡 갱신패턴 설명:
- 짝수시: 0시, 2시, 4시, 6시... (2시간 간격)
- 홀수시: 1시, 3시, 5시, 7시... (2시간 간격)
- 갱신분: 해당 시간의 몇 분에 갱신되는지

🔔 알림 설정:
- 브라우저에서 알림 권한 허용 권장
- 탭을 닫으면 알림이 작동하지 않음
- 항상 브라우저 탭을 열어두세요

📱 포터블 사용:
- 폴더 전체를 USB나 클라우드에 저장
- 다른 컴퓨터에서도 동일하게 사용 가능
- 인터넷 연결 불필요 (오프라인 작동)

🛠️ 문제 해결:
- CSV 파일이 안 열릴 때: UTF-8 인코딩으로 저장
- 알림이 안 올 때: 브라우저 알림 권한 확인
- 시간이 안 맞을 때: 컴퓨터 시간 확인

📞 업데이트나 문의사항이 있으면 말씀해주세요!
